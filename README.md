# open_thermostat

A lightweight, open-source thermostat controller. Based on Lolin S2 mini board

## Quickstart

Install
```bash
git clone https://github.com/your-org/open_thermostat.git
cd open_thermostat
python -m venv .venv
source .venv/bin/activate
esphome run open_thermostat.yaml
```
